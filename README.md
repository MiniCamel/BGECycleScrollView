## Installation

BGECycleScrollView is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
platform :ios, '9.0'

target '<#TargetName#>' do

pod 'BGECycleScrollView', '~> 1.0.0'

end
```
